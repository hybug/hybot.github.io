1. Linux统计文件夹下的文件数目

- 统计当前目录下文件的个数（不包括目录）
```
$ ls -l | grep "^-" | wc -l
```
- 统计当前目录下文件的个数（包括子目录）
```
$ ls -lR| grep "^-" | wc -l
```
- 查看某目录下文件夹(目录)的个数（包括子目录）
```
$ ls -lR | grep "^d" | wc -l
```

2. tar命令
压缩hello为test.tar.gz
```
tar -czvf test.tar.gz hello
```
解压
```
tar -xzvf test.tar.gz
```
